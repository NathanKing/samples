Different ways of viewing data



Matrix Data

Matrix data can only be viewed in the standard heatmap (bargraph) form.

GFF3 Data

GFF3 data can be viewed by chromosome or by feature.


How to switch between views

1. Click on the View menu item in the top left corner of the application window.
2. Click on "By Chromosome" to switch to the chromosome view.
3. Click on "By Feature" to switch to the feature view.
*Note*: Some program features may be disabled when using the feature view.
